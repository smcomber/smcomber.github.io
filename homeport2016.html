Create pages

1

Select the gear icon in the upper-right. In the drop-down, select Add a page.

2

Enter the exact name of the existing Homeport page. If creating a new page, use all lowercase characters, dashes instead of spaces, and no special characters. Select Create.

3

In the edit ribbon, select Edit Properties

4

Update the Title field to something user-friendly. Select Save.

5

The title will display in the gray bar below the logo and site name.

6

Your page is now ready for content!

To get started, select the Page tab, then Edit
